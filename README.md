# Youtube-Streamer

Searches for the youtube video and streams it instantly.


# Installation
```
$ pip install youtube-streamer
```

# Usage
Query for the video:
```
$ ys video_player_name [Video/Keyword]
```

```
$ ys vlc shape of you
```
